如何搭建属于自己的静态博客

本文最后更新于：3 months ago

基于Windows下的搭建hexo框架博客

博客这个东西对于我们从事IT行业的人来说其实很重要的，不管是以后出去面试，还是学校招聘都是很加分的，所以决定搭建一个，记录一下我的生活和学习。
第一次搭建静态博客，动态博客之前我也搭建过的，相比之下，我觉得静态博客会更好操作一点，毕竟动态博客还要买服务器，备案啥的，很不方便。
但如果你喜欢多折腾的话，动态博客也是可以的。下面记录一下我的“第一次”，想想还有有点小羞涩。

1、安装Node.js

去这里安装 Node.js

简单的说 Node.js 就是运行在服务端的 JavaScript。
Node.js 是一个基于Chrome JavaScript 运行时建立的一个平台。
Node.js是一个事件驱动I/O服务端JavaScript环境，基于Google的V8引擎，V8引擎执行Javascript的速度非常快，性能非常好。

Node.js是搭建静态博客必须要用的

2、安装博客框架

首先需要借助npm包管理器来安装，但国内的镜像源速度很慢，所以这里我们利用npm来安装cnpm(淘宝的那个源)

win+R，搜索cmd，以管理员身份运行下面代码

npm install -g cnpm --registry=https://registry.npm.taobao.org

安装完了以后接着安装hexo博客框架

cnpm install -g hexo-cli

安装完了以后

hexo -v
来查看hexo的信息

接着建立一个文件夹，mkdir blog
在文件夹里面利用hexo来建立博客hexo init(初始化一个博客)
这里你可能会出现一系列问题：

1、Git没有安装好或者配置好，尤其是username和email；
2、安装好了，环境变量没有配置；

这些其实网上都能查得到的，不要觉得烦，不耐烦说明你就不适合这个

3、操作

3-1如何启动博客
打开Git CMD ，输入 hexo s 启动博客 在你的浏览器url栏里面输入 http://localhost:4000 就可以看到你的博客了
3-2如何写博客

先CTRL+C断掉连接
在命令行里面输入 hexo n "我的第一篇博客文章" 创建一个新的博文
Windows下在你的 blog\source_posts 文件中就可以发现这篇博文
你可以用Sublime这个工具打开，我用起来也是很顺手的，当然md文件建议用Typora来打开
然后就可以自己操作了，这个就懂得都懂了
